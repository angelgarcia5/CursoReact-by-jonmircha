Que es un evento?

Los eventos son acciones del usuario que React puede detectar y manejar


Los eventos mas comunes a usar :
| Evento                                 | Tipo de interacción              | Dónde se usa                  | Qué detecta                                                        |
| -------------------------------------- | -------------------------------- | ----------------------------- | ------------------------------------------------------------------ |
| `onClick`                              | Clic del mouse o toque           | Botones, divs, imágenes, etc. | Cuando el usuario **hace clic** en el elemento                     |
| `onChange`                             | Cambio en valor de entrada       | Inputs, selects, textareas    | Cuando el usuario **modifica** el valor                            |
| `onSubmit`                             | Envío de formularios             | `<form>`                      | Cuando el usuario **envía** un formulario                          |
| `onMouseEnter` / `onMouseLeave`        | Movimiento del cursor            | Cualquier elemento            | Cuando el mouse **entra o sale** de un área                        |
| `onKeyDown` / `onKeyUp` / `onKeyPress` | Teclado                          | Inputs o la ventana           | Cuando el usuario **presiona o suelta** una tecla                  |
| `onFocus` / `onBlur`                   | Enfoque de campos                | Inputs o botones              | Cuando el elemento **gana o pierde foco**                          |
| `onInput`                              | Cambios en inputs en tiempo real | Inputs                        | Detecta **cada cambio** en el campo (más inmediato que `onChange`) |
